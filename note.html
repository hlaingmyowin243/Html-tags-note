Heading

Syntax:

<h1>Title</h1>

Example:

<h1>Welcome to HTML</h1>
Paragraph

Syntax:

<p>Text here</p>

Example:

<p>This is a paragraph.</p>
Span

Syntax:

<span>text</span>

Example:

<p>This is <span>important</span></p>
Image

Syntax:

<img src="image.jpg" alt="desc" />

Example:

<img src="cat.jpg" alt="A Cat" />
Anchor

Syntax:

<a href="url">Link</a>

Example:

<a href="https://google.com">Google</a>
List

Ordered:

<ol><li>Item 1</li></ol>

Unordered:

<ul><li>Item 1</li></ul>
Bold & Italic

Syntax:

<b>text</b>, <strong>text</strong> <i>text</i>, <em>text</em>

Example:

This is <b>bold</b> and <i>italic</i>
Mark

Syntax:

<mark>highlight</mark>

Example:

This is <mark>important</mark>
Delete

Syntax:

<del>removed</del>

Example:

<del>Old Price</del>
Sub & Sup

Syntax:

H<sub>2</sub>O, x<sup>2</sup>

Example:

H2O, x2
Select

Syntax:

<select> <option>One</option> </select>

Example:

<label>Choose:</label> <select><option>Red</option></select>
Iframe

Syntax:

<iframe src="url"></iframe>

Example:

<iframe src="https://example.com" width="300" height="200"></iframe>
Video

Syntax:

<video controls> <source src="video.mp4" type="video/mp4" /> </video>

Example:

<video controls width="320" height="240"> <source src="movie.mp4" type="video/mp4" /> </video>
